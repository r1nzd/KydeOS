# KydeOS

A personal operating system based on Debian, customized with a Material 3 UI.

##  Key Features
-  **Material 3 UI:** Comprehensive customization providing a unified and modern visual experience.
-  **Performance Optimization:** Stripped of unnecessary components, delivering smooth operation.

##  Build & Test Guide (For Devs)
1. Environment: WSL2 (Debian)
2. Direct customization on rootfs.
3. Export and test on QEMU.
